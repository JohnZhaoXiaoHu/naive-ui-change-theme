# Naive UI 动态自定义换肤实现

具体说明参考：<https://liubing.me/article/vue/naive-ui/naive-ui-custom-theme.html>

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm run dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm run lint
```
